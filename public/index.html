Quran API Documentation

This API provides access to Quran tafsir and translation text files. Please note that all files are compressed using BZip2. You will need to decompress the content within your application.

API Usage

To access any file, you'll use a specific URL format. Replace http://localhost:3000 with the actual domain name of your deployed application.

Accessing Files

Translation Files

Translation files are located in the /translation directory. To access a specific translation file, use the following URL format:

http://localhost:3000/translation/.txt

Replace with the ID of the translation file you want to access (e.g., 19 for file 19.txt).

For example, to get the 19.txt file, use this URL:

http://localhost:3000/translation/19.txt

Tafsir Files

Tafsir files are located in the /tafsir directory. To access a specific tafsir file, use the following URL format:

http://localhost:3000/tafsir/.txt

Replace with the ID of the tafsir file you want to access (e.g., 14 for file 14.txt).

For example, to get the 14.txt file, use this URL:

http://localhost:3000/tafsir/14.txt
